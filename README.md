高效，是贯穿蓝湖所有工作的高频词，不仅产品追求高效，我们团队的工作节奏也同样高效，没有冗长的会议、没有繁琐的交付流程、更没有复杂的层级管理，以结果为导向，让你专注做事高效产出。

成长，是每一位蓝湖人最深刻的体验，无论是技术能力的成长，还是个人观念的提升，你的成长需求总能得到到恰当的指导，季度之星、绩效考核、年终奖励都以个人的成长为主要标准。
